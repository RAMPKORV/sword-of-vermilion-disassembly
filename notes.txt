Map format:
; NOTE: Canonical documentation is in src/mapdata.asm header (full overworld/cave/town breakdown).
Map data is RLE encoded.

You read a byte (A)
If < $80, output it as is
If >= $80, read another byte and output it ($80 - A) times.
Repeat

Tile values:
$00 => Ground
$01 => Tree
$02 => Rock
$05 => Cave rock
$0F => House 

$8x => Entrance to town nr x
$1x => Entrance to cave nr x or lower level of cave
$FF => Cave exit

Cursor_selected_option  = $FFFFC23E     ; Menu_cursor_index (constants.asm / game_constants.asm)
